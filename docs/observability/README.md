# 可观测性概述

本目录包含可观测性栈的配置和文档，支持对 AI 系统进行全面监控。

## 三支柱

```
┌─────────────────────────────────────────────────┐
│               Observability                      │
├───────────────┬───────────────┬─────────────────┤
│    Metrics    │    Traces     │      Logs       │
│   (指标)      │   (追踪)      │     (日志)      │
├───────────────┼───────────────┼─────────────────┤
│ • 延迟        │ • 请求路径    │ • 结构化日志    │
│ • 错误率      │ • 调用链      │ • 错误追踪      │
│ • 吞吐量      │ • 依赖关系    │ • 审计日志      │
│ • 饱和度      │ • 性能瓶颈    │ • 变更日志      │
└───────────────┴───────────────┴─────────────────┘
```

## 技术栈

| 组件 | 工具 | 用途 |
|------|------|------|
| 指标收集 | Prometheus | 时序数据存储和查询 |
| 可视化 | Grafana | 仪表盘和告警 |
| 分布式追踪 | OpenTelemetry + Jaeger | 请求追踪和性能分析 |
| 日志聚合 | Loki | 日志收集和查询 |

## 快速开始

```bash
# 启动可观测性栈
docker-compose -f observability/docker-compose.yml up -d

# 访问仪表盘
open http://localhost:3000  # Grafana
open http://localhost:16686 # Jaeger
```
